# Classy Watch Pack

A collection of my favorite timepieces replicated in Samsung Watch Studio, with battery-life optimization.

## Available watches:

- Omega Seamaster 2900.50.91 (from the movie Casino Royale)
- Omega Speedmaster 145.012-67 (from the Apollo 11 Moon Landings)
- Omega Skywalker X-33

## How to install

Use ADB to install these onto your Wear OS 3.0+ Smartwatch. To learn how to do that, click here.
